# Synthetic Data Guidelines

## Default stance

Do not generate synthetic data unless it is clearly safe and fully detached from real records.

## If synthetic material is created later

- create it from scratch
- keep it small
- avoid real distributions
- avoid real institutions
- avoid real identifiers
- avoid real dates and sequences
- avoid free-text copied from real records

## If there is any doubt

Do not create synthetic material.
Use a TODO file instead.
