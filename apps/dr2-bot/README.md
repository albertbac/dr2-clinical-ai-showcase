# Evidence-first Specialist Analyst Assistant

## Summary
A document-aware analytical assistant that combines retrieval, structured ingestion, and audit-friendly evidence panels.

## Primary user
Specialist analyst

## Secondary user
Clinical or knowledge operations reviewer

## Problem solved
Analysts need evidence-backed answers across case material and reference material without losing traceability.

## Functional workflow
Documents are ingested and indexed, retrieval combines multiple evidence sources, and the user receives an answer paired with supporting evidence and processing status.

## Public-safe technical scope
Public-safe scope: ingestion pipeline, evidence-first response model, administrative controls, and high-level retrieval architecture.

## High-level integrations
High-level only: literature sources, optional graph layer, and managed document storage.

## What stays private and why
Prompt templates, ranking weights, provider configuration, routing rules, and operational storage structure remain private.

## Confidence level
HIGH

## Exposure risk
MEDIUM
