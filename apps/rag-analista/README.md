# RAG Analyst for Document-heavy Operations

## Summary
A multi-tenant analysis platform that answers over document context with retrieval, guided prompts, and controlled operational review.

## Primary user
Operational analyst

## Secondary user
Knowledge or administrative reviewer

## Problem solved
Document-heavy teams need structured answers across uploaded files and administrative knowledge without losing control over context.

## Functional workflow
Users upload files, run open or guided analysis flows, and receive answers grounded in retrieved context from controlled knowledge sources.

## Public-safe technical scope
Public-safe scope: RAG workflow, guided-question mode, administrative knowledge management, and multi-tenant boundaries.

## High-level integrations
High-level only: identity, vector retrieval, object storage, and controlled model providers.

## What stays private and why
Provider routing, operational prompts, retrieval configuration, ranking choices, and deployment wiring remain private.

## Confidence level
HIGH

## Exposure risk
MEDIUM
