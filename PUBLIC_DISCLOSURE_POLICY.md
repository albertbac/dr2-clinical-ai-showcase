# Public Disclosure Policy

## Principle
Minimal exposure. Strong documentation. Zero avoidable leakage.

## What the hub publishes
- the thesis of the public-safe showcase
- categorized links to the app repositories
- publication status and navigation guidance

## What the hub does not publish
- production code
- app internals
- raw records
- private infrastructure detail
- screenshots by default

## Publication rule
If a repository falls outside the closed app list or introduces material doubt, it should not be linked from the hub.
